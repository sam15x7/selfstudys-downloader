browser extension
Tampermonkey is the industry-standard extension for running custom scripts in your browser. It's open-source, trusted by 10M+ developers, and available on every major browser. Pick your browser from the grid above and click install. Takes 15 seconds.
02
Install the Script
one-click
Click the big → INSTALL SCRIPT button at the top of this page. Tampermonkey intercepts the .user.js file and opens its install dashboard automatically. Hit the green Install button inside that tab. Done — the script is live.

Go Get Your Papers
profit
Open any paper on selfstudys.com. Look at the bottom-right corner of your screen — there's a frosted-glass animated dock waiting for you. Hit Download. The PDF saves instantly with the correct title. No renaming. No copy-pasting URLs. No nonsense.
// under the hood

What it
actually does.

⚡
Viewer Bypass
Surgically extracts the real PDF URL hidden beneath Adobe Embed and Flipbook wrappers. No iframes, no nonsense.
🏷
Auto-Naming
Reads the page title and names the file correctly so your Downloads folder stays sane. Every. Single. Time.
🎨
Frosted Dock UI
Injects a minimal, animated download dock directly on the page. Doesn't touch or break any existing UI element.
🔒
Zero Telemetry
Runs 100% client-side. No servers, no tracking, no requests outside selfstudys.com. Your data stays yours.
